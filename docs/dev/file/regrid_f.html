regrid.F90¶

regrid_base_module¶

Description

Regrid base state variables (ex. psi, etarho, rho0, etc.)

We copy the coarsest level only, interpolate to all the other levels and then copy the valid data from the old arrays onto the new.

Quick access

Routines

regrid_base_state_cc(), regrid_base_state_edge()

Required modules

  • amrex_mempool_module (bl_allocate(), bl_deallocate())

  • amrex_constants_module

  • base_state_geometry_module (nr_fine(), dr(), nr(), max_radial_level(), numdisjointchunks(), r_start_coord(), r_end_coord(), finest_radial_level()): a module for storing the geometric information so we don’t have to pass it
