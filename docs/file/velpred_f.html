velpred.F90¶

velpred_module¶

Description

velpred is called by advance_premac – it is used to predict the normal velocities to the interfaces. We don’t care about the transverse velocities here. The prediction is done piecewise linear (for now)

Quick access

Routines

velpred_2d(), velpred_3d(), velpred_interface_2d(), velpred_interface_3d(), velpred_transverse_3d()

Required modules

  • amrex_error_module

  • amrex_mempool_module (bl_allocate(), bl_deallocate())

  • amrex_constants_module

  • slope_module

  • ppm_module: compute the PPM integrals, Ip and Im. These are the integrals under the parabolic profile of the reconstructed quantity over the domain that can reach the interface over the timestep dt….

  • meth_params_module (ppm_type(), rel_eps(), spherical(), ppm_trace_forces())

  • base_state_geometry_module (nr_fine(), max_radial_level()): a module for storing the geometric information so we don’t have to pass it
